Keamanan adalah Prioritas ✅

SidraWallet dirancang dengan pendekatan self-custody. Seed phrase dan private key hanya tersimpan di perangkat Anda. Jangan pernah membagikan seed phrase kepada siapa pun.

  • ✔ Data wallet disimpan di perangkat Anda
  • ✔ Tidak menggunakan server untuk menyimpan seed phrase
  • ✔ Open source dan dapat diaudit oleh siapa saja
  • ✔ Tidak ada analytics atau tracking apapun
  • ✔ Tidak perlu registrasi/akun
  • ✔ Baca Privacy Policy lengkap

Tentang Sidra Wallet

Sidra Wallet adalah dompet non-custodial open source yang dibangun untuk ekosistem Sidra. Dikembangkan oleh Atos, aplikasi ini memberikan kendali penuh atas seed phrase dan aset Anda langsung dari perangkat Android, tanpa server pihak ketiga yang menyimpan data sensitif Anda.

Pertanyaan Umum

Apakah Sidra Wallet aman?

Ya. Seed phrase dan private key hanya tersimpan di perangkat Anda, tidak pernah dikirim ke server manapun. Kode sumbernya terbuka dan dapat diaudit siapa saja.

Bagaimana cara menginstal aplikasinya?

Unduh file APK terbaru dari halaman GitHub Release resmi, lalu instal langsung di perangkat Android Anda. Pastikan mengizinkan instalasi dari sumber tidak dikenal di pengaturan HP.

Apakah wallet ini gratis?

Ya, Sidra Wallet sepenuhnya gratis dan open source, dikembangkan untuk komunitas ekosistem Sidra.
